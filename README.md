# Alien Invasion Game

An **alien invasion game** built with Python and Pygame.

This project demonstrates event handling, sprite movement, collision detection, game loops, and basic game mechanics.

## Technologies Used

| Technology | Purpose |
|------------|---------|
| Python     | Game logic |
| Pygame     | Graphics and interaction |

## How to play

1. **Clone the repository**

```bash
git clone https://github.com/mahikachadha5/alien-invasion.git
cd alien-invasion
```

2. Create a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install pygame

```bash
pip install pygame
```

4. Run and play!

```bash
cd alien-invasion
python alien_invasion.py
```

## Game Rules

### Goal
To shoot all alien ships before they take over your spaceship.

### How to play
- to move your spaceship, use the <kbd><--</kbd> and <kbd>--></kbd> arrow keys
- to quit, press <kbd>q</kbd>
- to shoot, press <kbd>spacebar</kbd>
